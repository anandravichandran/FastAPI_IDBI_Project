# =============================================================================
# Financial Suite - production dependencies (PINNED for reproducible builds)
# =============================================================================
# TARGET RUNTIME: CPython 3.12  (see runtime.txt / .python-version / Dockerfile)
#
# Every package below publishes prebuilt manylinux **cp312** wheels, so a clean
# `pip install -r requirements.txt` NEVER compiles native code:
#   * no Rust/maturin build of `tokenizers`
#   * no source build of torch / numpy / pydantic-core / grpcio / onnxruntime
#
# Do NOT install on Python 3.13/3.14: several of these wheels are not published
# for those ABIs yet, so pip would fall back to source builds (the exact Render
# failure this pinning fixes). The Docker image pins python:3.12-slim and sets
# PIP_ONLY_BINARY so a missing wheel fails fast instead of invoking Rust.
# -----------------------------------------------------------------------------
